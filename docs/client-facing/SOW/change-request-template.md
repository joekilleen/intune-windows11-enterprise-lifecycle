## 1. Description of Requested Change

Provide a clear and concise description of the requested change.

- What is being added, removed, or modified?
- Which existing deliverable, phase, or assumption is affected?
- Why is the change being requested?

> _Example:_  
> Add support for Windows Autopilot pre-provisioning (White Glove) for shared devices, which was not included in the original scope.
