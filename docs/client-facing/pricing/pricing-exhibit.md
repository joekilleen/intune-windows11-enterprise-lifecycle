# Pricing Exhibit  
**Windows 11 Enterprise Device Lifecycle Implementation (Microsoft Intune)**

---

## 1. Purpose

This Pricing Exhibit defines the **commercial terms and fixed-fee pricing** associated with the Windows 11 Enterprise Device Lifecycle implementation described in the Statement of Work (“SOW”).

The intent of this exhibit is to:
- Provide transparent, predictable pricing
- Align cost to clearly defined outcomes
- Support procurement and budget approval
- Minimize scope and pricing disputes

All pricing is governed by the Master Services Agreement (“MSA”) and SOW.

---

## 2. Pricing Model

### 2.1 Model Type

☑ **Fixed Fee**  
☐ Time & Materials (Not Applicable)

The fixed-fee model assumes:
- Defined scope
- Agreed assumptions
- Client participation as specified in the SOW

Any deviation from scope is governed by the **Change Request process**.

## 4. Deliverables Included

The fixed fee includes delivery of:

- Enterprise-grade architecture documentation
- Production-ready Intune configuration guidance
- Operational runbooks
- Governance and RBAC framework
- CIS / NIST / ISO compliance crosswalk
- Evidence mapping matrix
- Executive summary and one-page brief

All deliverables are provided in **client-owned Markdown format**.

---

## 5. Payment Schedule

Unless otherwise specified in the SOW, payment is structured as follows:

| Milestone | Percentage | Amount |
---|---:|---:|
| Engagement kickoff | 30% | $17,550 |
| Architecture & core configuration complete | 40% | $23,400 |
| Final delivery & acceptance | 30% | $17,550 |
| **Total** | **100%** | **$58,500** |

Invoices are issued per milestone and payable per MSA terms.

---

## 6. Pricing Assumptions

This pricing assumes:

- Single Microsoft Entra ID tenant
- Single Intune tenant
- Windows 11 corporate-owned devices
- One primary production environment
- One non-production / pilot environment
- Up to:
  - 5 security baselines
  - 10 compliance policies
  - 20 Win32 application designs
- Client-provided licensing
- Remote delivery

Additional complexity may require a change request.
